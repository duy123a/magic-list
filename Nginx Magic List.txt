I. For local


1. Install nginx locally (pls don't use docker for this one, it will make it more complex)

- Install at C:\nginx
- Add PATH


2. Create C:\nginx\conf\sites-enabled\

Modify C:\nginx\conf\nginx.conf to something like this

#user  nobody;
worker_processes  1;

events {
    worker_connections 1024;
}

http {
    include       mime.types;
    default_type  application/octet-stream;

    sendfile        on;
    keepalive_timeout 65;

    # include all virtual host
    include C:/nginx/sites-enabled/*.conf;
}

3. Install OpenSSH

- Download from Shining Light Productions

Where should the OpenSSL DLLs be copied? => The OpenSSL binaries (/bin) directory

- Add PATH: C:\Program Files\OpenSSL-Win64\bin

- Test: openssl version


4. Get self cert from .NET

- Trust cert

dotnet dev-certs https --trust

- Export cert file for nginx

dotnet dev-certs https -ep C:\nginx\certs\example.local.pfx -p 123456

- Convert PFX → PEM (nginx need)

openssl pkcs12 -in example.local.pfx -out cert.pem -nodes (password 123456)
openssl rsa -in cert.pem -out cert.key


5. Setup example.local.conf in sites-enabled

server {
    listen 80;
    server_name example.local;

    return 301 https://$host$request_uri;
}

server {
    listen 443 ssl;
    server_name example.local;

    ssl_certificate     C:/nginx/certs/cert.pem;
    ssl_certificate_key C:/nginx/certs/cert.key;
	
    # =========================
    # Upload settings (future)
    # =========================
    client_max_body_size 50M;

    location /auth/ {
		proxy_pass http://localhost:5000;
		proxy_set_header Host $host;
		proxy_set_header X-Forwarded-For $remote_addr;
		proxy_set_header X-Forwarded-Proto https;
		proxy_set_header X-Forwarded-Host $host;
		proxy_set_header X-Forwarded-Port 443;
		# (Optional) SignalR / WebSocket support
		proxy_set_header Upgrade $http_upgrade;
		proxy_set_header Connection "Upgrade";
    }

    location /main/ {
		proxy_pass http://localhost:5001;
		proxy_set_header Host $host;
		proxy_set_header X-Forwarded-For $remote_addr;
		proxy_set_header X-Forwarded-Proto https;
		proxy_set_header X-Forwarded-Host $host;
		proxy_set_header X-Forwarded-Port 443;
    }
}


6. Start nginx (using cmd in C:\nginx, for some reason PATH not work for me)

start nginx

nginx -s reload 

nginx -s stop 

nginx -t 

taskkill /F /IM nginx.exe
